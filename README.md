# 🧾 Form Validator PRO

A modern, interactive form validation project built with HTML, CSS and JavaScript.

This project focuses on creating a clean, professional user experience with real-time validation, smooth UI feedback, and reusable front-end logic.

## ✨ Features

* 🔥 Live input validation (real-time feedback)
* ❌ Error states with messages
* ✅ Success states with visual feedback
* 👁️ Show / Hide password toggle
* 📊 Password strength bar
* 💾 LocalStorage (auto-save form data)
* 🎬 Custom success modal (no default alert)
* 🚫 Disabled submit button until valid
* 🎨 Modern dark UI design
